# dewu-ai-shopping-guide-demo

这是一个手机页面视角的 `AI潮流导购` 静态原型，用于模拟年轻用户从输入需求、确认理解、补充偏好，到查看推荐、进入模拟详情、对比方案、生成搭配和轻量自评的核心流程。

## 项目特点

- 使用 `HTML`、`CSS`、`JavaScript` 完成，无需复杂框架。
- 支持直接双击 `index.html` 在浏览器中运行。
- 页面风格为白底克制的潮流电商界面，使用青绿色品牌识别，并强调正品安心和搭配参考。
- 识别页支持用户修改系统理解的信息，避免误解继续传递到推荐结果。
- 追问页默认只展示快捷选项，点击“其他想法”后才展开自填输入，并用关键词理解自填内容。
- 推荐卡会根据用户补充的顾虑，动态调整价格、尺码、上身和撞款相关提醒。
- 推荐方向会根据用户的推荐倾向动态排序，并标记“更贴近本轮想法 / 备选 / 再比较”。
- 点击推荐图可进入对应模拟详情页，查看方向说明、场景、尺码建议和正品安心提示。
- 推荐卡支持加入对比清单，对比页会高亮已加入项。
- 推荐卡支持继续追问，可围绕尺码、上身、撞款、通勤、预算等问题生成模拟回答。
- 所有推荐都使用“方向”表达，不编造具体购买信息。

## 演示流程

1. 首页展示 `AI潮流导购`、副标题、输入框和快捷需求按钮。
2. 用户输入购物需求并点击“开始导购”。
3. 页面展示可编辑的信息确认页，用户可修改大致范围、方向、场合、服务对象和希望保留的感觉。
4. 用户补充购买顾虑和推荐倾向，也可点“其他想法”自填。
5. 页面横向展示三张单品方向卡片：简约德训鞋、复古跑鞋、基础板鞋；每次滑动聚焦一个方向。
6. 用户可点击推荐图进入模拟详情页，也可加入对比清单或继续追问。
7. 用户可以点击“对比方案”查看表格，也可以点击“生成搭配”查看通勤、约会和提醒。
8. 用户结束导购后进入低成本自评，可按多个维度打分，也可跳过。

## 推荐输入

```text
预算800以内，想买一双适合通勤和约会的男生休闲鞋，风格不要太夸张
```
